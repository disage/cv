Cooking Companion

Date of creartion: 06.2023

Stack: React JS

Additional Technologies: Firebase, Chat GPT

Description

Сooking Сompanion is React application which designed to assist users in creating a meal using ingredients they have on hand. Upon launching the application, users are greeted with a user-friendly interface where they can enter the ingredients they want to include in their dish. Application use Chat GPT API to generate recipes and cooking instructions. Google Firebase is used to store user products.
